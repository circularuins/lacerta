# Lacerta — Concept

## Vision

目標を入力すると、AI が逆算して毎日のタスクを自動生成する。進捗に応じて計画を再計算し、使い込むほどユーザーのペースを学習して精度が上がる。

## Core Loop

1. **Goal** — ユーザーが目標と期日を入力（例: "2027年2月 グラミー賞受賞"）
2. **AI Plan** — ジャンル別テンプレートに基づいて AI が逆算タスクを生成
3. **Daily Tasks** — 今日やるべきタスクをプッシュ通知で配信
4. **Progress** — 完了/部分完了/スキップを入力
5. **Recalculate** — AI が翌日以降の計画を再計算
6. **Learn** — ペースデータを蓄積し、次のプラン精度を向上

## Genre Templates

ジャンルごとに必要な工程を定義。AI はテンプレートを参照しつつ、期日とペースに合わせてタスクを配分する。

### 例: 作曲

1. アイデア出し・リファレンス収集
2. 作詞
3. メロディー作成
4. コード進行
5. DTM（打ち込み）
6. Mix / Mastering
7. 配信ディストリビューション作業

## Named after

Greg Egan "Diaspora" の **Lacerta 信号** — 遠い目標（信号）を検知し、そこへの到達経路を逆算するという物語の構造が、このアプリの本質と一致する。
